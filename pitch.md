Problem Statement

The old menu-based portal is inefficient because it relies heavily on manual navigation, making it time-consuming and confusing for users to find information. It lacks personalization, intelligent assistance, and automation, which leads to user frustration, slower task completion, and reduced productivity.

Industry 5.0 Justification

Industry 5.0 focuses on human-centered technology where AI enhances human capabilities instead of replacing them. In this system, AI assists users by providing smart recommendations, automating repetitive tasks, and simplifying decision-making. This empowers users to work faster, make better choices, and have a more personalized experience while still maintaining full control.

Agile Requirements (User Stories)
As a user, I want an AI-powered search assistant, so that I can quickly find the information I need without navigating multiple menus.
As a user, I want personalized recommendations, so that I can access relevant features and content more efficiently.
As an admin, I want AI-based analytics and insights, so that I can make better decisions and improve system performance.
